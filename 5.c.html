CSS Font & Text

Font size

The CSS font-size property is used to specify the size of the text/font.

Font-weight

The font-weight property sets how thick or thin characters in text should be displayed.

Font-style

The font-style property specifies the font style for a text. Font-style like normal,italic,oblique,

Text-decoration

The text-decoration property specifies the decoration added to text. text-decoration are underline,overline,line-through.

Text-transformation

The CSS text-transform property is used to control the capitalization of text in an element.

Text-alignment

The text-align property is used to set the horizontal alignment of a text.
